# Javascript---Exercise-4-Collabo-
We both wrote our codes individually, then we compared and chose the cleaner code between the two.


This JavaScript project was created as a collaboration between two developers to practice using variables and conditional statements. The program checks a student's score, compares it to the passing grade, and displays whether the student achieved a perfect score, passed, or failed. Through this project, we improved our understanding of JavaScript basics, teamwork, problem-solving, and writing clean and simple code.
